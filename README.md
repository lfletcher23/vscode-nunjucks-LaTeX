# This is a highlighter for a custom Nunjucks syntax intended to be compatible with LaTeX

## Nunjucks syntax:
variableStart: #!

variableEnd: !#

blockStart: [##

blockEnd: ##]

commentStart: %!

commentEnd: !%

### Supports these file extensions
```
.njk, .nj, .nunjucks
```
